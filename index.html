Raleigh Install Services SOP

Framing SOP

Overview

These procedures outline the full lifecycle of a framing job from pre-check through closeout. Appwright is the primary system used for scheduling, communication, and task tracking. All required tasks must be completed and signed off within Appwright.

Job Pre-Check (Appwright Task)
  • Verify site ready within 2 days
  • Site graded, backfilled, safe for framing
  • Temp power available (document if generator needed)
  • Foundation ready
  • Plans in permit box
  • Measure pony walls and send to sales
  • Order steel if possible
Delivery Coordination:
  • Basement / 1st floor framing
  • Wall panels
  • 2nd floor framing
  • Trusses

IMPORTANT: All schedule changes MUST go through Appwright.

Subcontractor Readiness
  • Confirm subcontractor assignment in Appwright
  • Call subcontractor directly to confirm start date
  • Verify crew is approved
  • Confirm delivery schedule
  • If TBD → contact Install Manager
Pre-Construction Meeting
  • Redline plans with builder
  • Upload redline plans in Appwright
  • Verify window & door openings
  • Confirm truss delivery
  • Confirm stair order
  • Identify change orders

Any major change requires approval and may require a change order before work begins.

Framing Start
  • Block foundation
  • Open floor inspection required
  • Coordinate inspection with builder
  • Slab must be square and level
  • Document all issues in Appwright
Project Manager Duties
  • Daily jobsite visits required
  • Update Appwright schedule
  • Conduct daily safety checks
  • Communicate with subcontractor, builder, and sales
  • Track material usage

Material Requests: Must include WHY and WHAT in Appwright.

Safety Inspections
  • 3 per week / 12 per month / 36 per quarter
  • Photos + notes required
  • PM + subcontractor must sign
  • Violations must be documented
First Walk Framing
  • Must be completed before framer leaves
  • Check ALL window & door openings
  • Mark incorrect openings with paint
  • Compare field vs plans
  • Required before 90% payout
Material Return Requests
  • Stack materials neatly
  • Upload photos
  • Send Appwright message to Sales + Framing
Second Walk Framing
  • Re-mark missed items
  • Identify trade damage
  • Extra work requires approval BEFORE starting
  • Check bowed studs
Framer Extras
  • Must be approved BEFORE work starts
  • Submit in Appwright
  • Builder PO required
  • No approval = no pay risk
Job Closeout
  • Final inspection passed
  • Upload inspection reports
  • Install goal posts
  • Triggers final 10% payout
Key Policies
  • No framing during severe weather
  • No truss install during high winds
  • All backcharges documented
  • All builder communication documented
Truss & Damage Documentation
  • Photos REQUIRED before install
  • No photos = no pay for repair
  • Framer owns damage after install
Equipment Rentals
  • All rentals go through Install Department
  • PM cannot schedule rentals
  • Provide equipment ID for service requests
