Framing SOP

Overview

The Framing SOP outlines the full lifecycle of a framing job from initial scheduling through final completion and closeout. This process ensures consistency, quality, safety, and accountability across all Builders FirstSource Raleigh Install Services framing operations.

All framing jobs must be managed through Appwright, which serves as the central system for scheduling, communication, documentation, and task tracking. Every Project Manager is responsible for maintaining accurate and up-to-date information within Appwright throughout the duration of the job.

The goal of this SOP is to:

  • Ensure all jobs are properly scheduled and staffed
  • Maintain consistent build quality across all projects
  • Enforce safety standards aligned with “Drive to Zero”
  • Provide clear communication between Install, Sales, Builders, and Subcontractors
  • Document all work, changes, and issues for accountability

This SOP should be followed on every framing job without exception. Any deviations, change orders, or issues must be documented and approved through the proper channels before work continues.

Appwright
1) Job Pre-Check – Framing (Appwright Task)

The purpose of this task is to start the scheduling process and check jobsite readiness. All job information (subcontractor, address, plans, truss documentation, etc.) is located in the Appwright worklist under the document section.

Site Assessment

  • Verify site ready within 2 days
  • Site graded and ready for deliveries
  • Site backfilled and safe for framers
  • If unsafe → contact builder for resolution
  • Unfilled porches must be blocked off
  • Temp power available (or request resolution)
  • Document if generator is needed
  • Foundation ready within 2 days
  • Measure pony walls / knee walls and send to Sales via Appwright
  • Ensure permitted plans are in permit box
  • Measure and order steel if applicable

Delivery Coordination

  • Basement / First Floor framing
  • Wall panels
  • Second floor framing
  • Second floor wall panels
  • Trusses

IMPORTANT: Send ALL schedule requests in Appwright. DO NOT call Sales or Framing for schedule changes.

Framing Pre-Construction Checklist

Job Pre-Check

Complete prior to Pre-Con with Framer & Builder Rep (B.R.)

  • Power onsite, or notify sales team that framer will need to supply generator
  • Grade complete and site is safe for framers
  • Slab/foundation square and level (visual inspection)
  • Material delivery scheduled appropriately for build-time and jobsite space
  • Confirm frame plans are onsite and match plans in Appwright
  • Measure and submit foundation step-downs, if applicable

Redline Plan Instructions

  • Redline hardcopy of builder supplied plans and have B.R. sign/date each page. Upload photos of redlined plans to Appwright labeled "Redline plan – Date".

Redline Annotations Must Include:

  • Selections marked
  • Fireplace openings marked
  • Exterior door openings marked
  • Stair railing options marked
  • Shower/tub options marked
  • Window and Door R.O. verified and marked (if not call-size)
  • Tray/coffer ceiling details marked
  • Attic access opening size/location verified (minimum 22.5" x 31")
  • Shower bench height and pitch verified and marked
  • Additional or modified chases marked
  • Stud spacing verified if different than 16" O.C.
  • Any slab/foundation discrepancies discussed with builder superintendent
  • Any custom changes confirmed by builder and documented
  • Redlined plans signed and dated by builder superintendent

Note:

Any redline changes AFTER Pre-Con must be updated, signed/dated by B.R., and re-uploaded into Appwright labeled "Redline plan – New Date". An AW message must be sent to notify the team.

7) First Walk Framing – (Appwright Task)

This task should be completed before the framer is finished and before trades are onsite. Deficiencies should be marked with black or red marker in the area of error for the subcontractor.

A FULL WALK IS REQUIRED including inspection of all structural, architectural, and quality control items. Refer to Section VI pages 1–4 for examples.

First frame walk must be completed prior to paying the framer their 90% draw.

The following must be verified:

  1. Checking all Window and Door openings – NO EXCEPTIONS
    • If correct → place check mark in opening
    • If incorrect:
      • Mark correct size in opening
      • Spray paint opening
      • Framer MUST fix before leaving
      • PM must verify correction
    • If BFS installs windows/doors:
      • PM must complete this task in Window & Door Appwright job
      • Must be completed before deadlines to prevent delays
    • Once complete → date and PM initials at front door
  2. Structural plan matches field construction
  3. Floor plan matches field conditions
    • Window & door counts, sizes, and locations correct
    • Selections installed correctly
  4. Elevation matches plans
  5. Trade prep items complete
    • Chases installed
    • Plywood at chases/flues
    • HVAC platforms (if required)

If truss layouts, seals, or documentation are missing:

  • Contact Framing Install Department
  • OR use “Job Lookup” in Appwright

9) Sub Completion Date – Framing (Appwright Task)

This step is completed when the framer has finished the project. The Project Manager is responsible for completing this task. This task triggers payment (Draw Payment 1) for framing through the Accounts Payable team in Appwright.

NOTE: Draw Payment 1 is not the responsibility of the Project Manager. It is strongly recommended to communicate completion expectations clearly to the subcontractor.

Completion includes the following:

  1. Jobsite cleanliness – inside and outside, including swept conditions
  2. 2x4 safety rails installed at:
    • Window openings
    • Stairway openings (handrails)
    • Any hazards
  3. Material stacked, organized, and ready for pickup
  4. Porch posts installed if required
  5. House wrap completed
    • Some builders require house wrap in window openings for inspections due to extended window lead times.
    • Verify builder-specific requirements before completion.
    • If house wrap in windows is intact at completion, photos are required for reimbursement of repairs later.

This task should NOT be signed off until the framing is fully complete.

It is strongly recommended that the Project Manager requires photo documentation from the subcontractor for all items listed above. This supports job cleanliness verification and provides documentation for future conflict resolution.

10) Second Walk Framing
  1. Remark any missed deficiencies from the First Walk using spray paint for subcontractor visibility.
  2. Identify and mark any damage caused by other trades for subcontractor clarity.
    • If additional work is required that was not agreed upon during the Pre-Construction Meeting OR is due to trade damage:
      • A labor pick ticket must be entered by the Sales team before work begins
      • Refer to Section VI, pages 9–11 for Extra Work Order processing
  3. Check for bad or bowed studs
    • This should be completed prior to sheetrock to prevent material movement issues
    • Refer to Section VI, page 13 for bowed stud policy
  4. Any additional material needed must be ordered at least 3 days prior to requiring subcontractor onsite
11) Framer Extras – (Appwright Task)

Framer extras must be entered into Appwright through a message sent to the Sales team and the Framing Install Department.

Before any work begins, written approval from the builder must be provided to the Sales team, ideally including a PO. Alternatively, the Sales team may provide approval to proceed.

No extra work should be started without approval. If work begins without approval, the framer risks not being paid.

Even if the Framing Appwright is closed out, all framer extras MUST still be entered into the Framing Appwright. The Framing Install Department or Sales team will create a Service Appwright to ensure payment is processed.

If a framer has questions regarding payment timing (90% / 10% draws or extras), they must contact the Framing Install Department. Sales teams do not determine payment timing.

12) Close Out Job – Framing (Appwright Task)

Once framing inspection has been completed and passed, the Project Manager must sign off on this task in Appwright. This ensures the framing subcontractor receives their final 10% of contracted labor.

  1. All inspection reports must be uploaded into Appwright under documents for record keeping.
  2. Goal posts must be installed at this time.
    • Do NOT wait for sheetrock installation.
    • If there are any questions or issues with builders regarding this requirement, they should be directed to the Install Manager.

If a project is no longer on your worklist, you can use Job Lookup to find it. There are several different ways to search for a project. The easiest ways are by Reference Number, Job ID#, or by keyword.

Step 2

Change the location you are searching under to include all BFS locations.

  • Select appropriate location filter
  • Enter job information (Job ID, address, or keyword)
  • Use search filters to narrow results if needed

Best Practice

  • Use Job ID when available for fastest results
  • Verify correct job before opening
  • Use keywords if full job info is unknown

Step 3

Change the search field categories based on the information you have available.

  • Select appropriate Search Fields (Subdivision, Lot, Builder Name, etc.)
  • Use "Starts With" or other filters to refine results
  • Enter known job details to narrow down the search

  • Search Field 1: Subdivision / Project → "Inwood"
  • Search Field 2: Lot / Bldg → "9"
  • Search Field 3: Builder Name → "Greenwood"

Best Practice

  • Use multiple search fields to narrow results faster
  • Avoid broad searches when possible
  • Confirm job details before opening

Reference

Step 4

After selecting "Search", every Appwright associated with the job will appear, including framing, truss, and panel jobs.

  • Review the list of returned Appwrights carefully
  • Identify the correct job based on job type and description
  • Multiple Appwrights may exist for the same address

Best Practice

  • Verify job type (Framing, Truss, Panels, etc.) before opening
  • Use job description and builder name to confirm accuracy
  • Do not assume the first result is correct

Framing Backcharges

Back Charge to Framing Install Department Policy

Below is the Builders FirstSource Raleigh Framing Market policy regarding back charges issued to the install department. If proper communication and documentation is not provided, BFS Framing Install Department reserves the right to deny the request for back charges.

  1. BUILDER Communication
    • Builder must notify the BFS Project Manager immediately upon discovery of any issue that may result in a back charge to BFS.
    • To the extent possible, builder must provide up to 48 hours from time of discovery for BFS to rectify or react to the allegation. BFS will make best effort to react/resolve within 24 hours.
    • If the issue absolutely cannot wait, BFS still needs to be notified immediately and given an opportunity to approve the alternate solution.
  2. BFS PROJECT MANAGER Responsibility
    • BFS Project Manager will assess the situation, document the situation (video and/or pictures), propose solution to builder for approval, initiate action plan with framer, and document final resolution (video and/or pictures) in Appwright.
  3. BUILDER Documentation
    • Courtesy email notification of impending back charges should be sent to BFS Sales Team within 24 hours of final determination of cause. Photos are strongly encouraged and required whenever BFS is unable to visit site.
    • Formal back charge document (invoice, work order, PO, etc.) should be submitted to BFS Sales Team within 60 days of work completed. This should include:
      • Final charge amount
      • Name of trade partner being paid
      • Description of work required by trade partner
      • Description of issue that caused the need for additional work
  4. BFS SALES TEAM Communication & Documentation
    • Forward all back charge email notifications immediately to Install Management and the respective Project Manager.
    • Review formal back charge document (invoice, work order, PO, etc.) upon receipt from builder to ensure criteria is met BEFORE passing to Install Sales for review. Request clarification from builder when necessary.
    • Send formal back charge document to Install Management in a timely manner (within 10 business days of receipt) for review and approval.

NOTE:

Shaft Wall Repair – Nails Through Shaft Liner

This procedure outlines the proper method for repairing damage caused by nails or fasteners driven through shaft wall liner panels in townhome construction.

Issue

  • Contractors have driven or inserted 16d nails into the 1" shaft liner panels
  • Creates penetrations that compromise fire rating and system integrity

Repair Procedure

  1. Remove nails from the shaft liner panels
  2. Fill holes with approved fire caulk (3M Fire Barrier Sealant CP 25WB+ or equivalent)
  3. Apply fire caulking on both sides of the wall where accessible
  4. Tool caulk smooth using putty knife or appropriate tool

Important Notes

  • ALL repairs must maintain fire-rated assembly integrity
  • Improper repairs can result in failed inspections
  • All damage and repairs must be documented in Appwright

Reference Repair Steps

  1. Nails driven through liner
  2. Nails removed – holes remain
  3. Holes patched with fire barrier sealant

Reference Detail

Shaft Wall Repair – Holes in Shaft Liner

This procedure outlines the proper repair for holes or penetrations through shaft liner panels to maintain the required fire-rated assembly.

Issue

  • Hole or penetration through 1" shaft liner panels
  • Compromises fire rating if not properly repaired

Repair Procedure

  1. Fill entire hole or penetration with approved fire caulk (3M Fire Barrier Sealant CP 25WB+ or equivalent)
  2. Cut a piece of 5/8" Type X drywall or approved fire patch material
  3. Center drywall patch over the hole
  4. Attach patch using drywall screws (minimum 1-1/4")
  5. Apply fire caulk around entire perimeter of patch
  6. If hole penetrates both sides, repeat patch on opposite side

Important Notes

  • Repair must maintain fire-rated wall integrity
  • All materials must meet ASTM / UL fire system requirements
  • All repairs must be documented in Appwright

Repair Steps Overview

  1. Identify penetration through shaft liner
  2. Fill hole with fire barrier sealant
  3. Install drywall patch with screws
  4. Seal perimeter with fire caulk

This condition applies to surface damage on shaft liner panels where the face paper or fiberglass mat is torn, scraped, or dented but does not fully penetrate the panel.

Shaft Wall Repair – Missing / Damaged H-Studs or C-Runner

This procedure outlines repairs required when H-studs or C-runner tracks are missing, damaged, or improperly installed in a shaft wall system.

Issue

  • Missing or damaged H-studs
  • Missing or damaged C-runner track
  • Gaps between framing members
  • Improper installation affecting fire-rated assembly

Repair Procedure

  1. Fill damaged or missing area with approved fire caulk (3M Fire Barrier Sealant CP 25WB+ or equivalent)
  2. Cut a metal flat strap (minimum 4" wide x 18" long)
  3. Center strap over damaged or missing section
  4. Secure strap to H-stud or C-runner using 1/2" Tek self-drilling screws
  5. Apply fire caulk around perimeter of metal strap
  6. If applicable, duplicate repair on opposite side of wall

Important Notes

  • All repairs must maintain fire-rated system integrity
  • Improper framing must be corrected immediately
  • All repairs must be documented in Appwright

Inspection Guidance

  • Verify framing continuity is restored
  • Ensure strap is securely fastened
  • Confirm all penetrations are sealed
  • Escalate major structural concerns to Install Manager

Shaft Wall Repair – Up to 6" Hole in Shaft Liner

This procedure applies to larger penetrations (up to 6") through shaft liner panels. These repairs require both internal liner patching and external drywall patching to maintain fire rating.

Issue

  • Penetration through shaft liner up to 6 inches
  • Compromises full fire-rated assembly if not properly repaired

Repair Procedure

  1. Cut two pieces of 1" shaft liner slightly smaller than the penetration
  2. Insert shaft liner pieces into cavity to fill void
  3. Laminate liner pieces together using approved fire sealant (3M Fire Barrier Sealant CP 25WB+)
  4. Secure laminated liner using 1-1/2" Type G laminating screws
  5. Apply fire caulk around edges of shaft liner repair
  6. Install 5/8" Type X drywall patch over opening
  7. Secure drywall patch using 1-1/4" drywall screws (max 6" spacing)
  8. Apply fire caulk around full perimeter of drywall patch
  9. If wall is accessible from both sides, duplicate repair on opposite side

Important Notes

  • Both liner AND drywall patch are required for proper fire rating
  • All materials must meet ASTM / UL fire system requirements
  • Improper repair will result in failed inspection
  • All repairs must be documented in Appwright

Inspection Guidance

  • Verify liner cavity is fully filled
  • Confirm drywall patch is secured properly
  • Ensure all edges are sealed with fire caulk
  • Check both sides of wall if accessible

    Plumb, Level, and Square

    If the two measurements are the same, the unit is square. If they are not the same, operational issues may occur.

    Window and Door Install Best Methods

    1. Flashing Sequence (MOST IMPORTANT)

    • Always follow: bottom → sides → top
    • Install:
      • Sill pan / flashing first
      • Set window into opening
      • Side flashing
      • Head flashing last
      • Do NOT caulk the bottom nail fin of the window

    Never reverse this order — causes water intrusion

    2. Check Opening Before Install

    • Verify:
      • Width & height
      • Square (diagonal check)
      • Level sill

    If opening is off → STOP and notify framing

    3. Set & Secure Window Properly

    • Center window in opening
    • Tack top corners first
    • Recheck:
      • Level
      • Plumb
      • Square

    Then fasten fully per manufacturer

    Flashing Method Reference

    4. Shim Correctly

    • Shim at:
      • Bottom corners
      • Midpoints
      • Top if needed

    Do NOT over-shim — causes bowing

    5. Operation Check (ALWAYS)

    • Open / close / lock every unit
    • Check:
      • Smooth movement
      • No binding
      • Locks engage properly

    DOOR INSTALL – BEST PRACTICES

    1. Start with the Hinge Side

    • Set hinge side perfectly plumb first
    • This controls the entire door alignment

    If hinge side is wrong → everything is wrong

    2. Proper Shimming (CRITICAL)

    • Shim:
      • Behind hinges
      • At strike plate
      • At top/bottom as needed
    • Strike plate area
    • Top corners

    NEVER:

    • Stack door pads
    • Shim randomly
    • “Force” the door into place

    3. Check Reveals

    • Even gap on all sides:
      • Top
      • Hinge side
      • Latch side

    Uneven reveal = frame not square

    4. Threshold & Seal

    • Ensure:
      • Threshold fully supported
      • Seal tight to floor
      • No daylight

    Water intrusion risk if done wrong
